## Camera Validation

1. Disconnect the SSD from the USB adapter
2. Install it onto the RED PCB
3. Insert into the camera
4. Power on and test

### Success Criteria

- No “Unauthorized Media” warning
- Media formats successfully
- Stable recording behavior
